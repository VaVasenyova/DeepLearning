7. Feature Importance Analysis

Understanding the Sigmoid Gate: The model includes a learnable gating layer that outputs importance weights (0-1) for each feature. Values close to 1.0 indicate the feature strongly influences predictions. Values close to 0.0 indicate the feature is largely ignored.

L1 Regularization: Applied to gate weights to encourage sparse feature selection, making the importance signals clearer.

Deployment Notes

To deploy this application:

  1. Create a public GitHub repository
  2. Commit index.html and app.js to the repository
  3. Enable GitHub Pages in repository settings (select main branch and root folder)
  4. Your app will be available at: https://[your-username].github.io/[repository-name]

Dependencies: TensorFlow.js and tfjs-vis are loaded from CDN, so no additional setup is needed.
